Overview

Mutual Fund Tracker is a web-based application designed to help users conveniently track their mutual fund investments. With this application, users can set target values for their selected mutual funds and receive email notifications when the target values match the corresponding mutual fund Net Asset Value (NAV).

Features

  • User Registration and Authentication: Create an account and log in securely.
  • Target Value Setting: Set desired target values for selected mutual funds.
  • Mutual Fund Data Retrieval: Fetch the latest NAV data from external sources.
  • Comparison and Notification: Notify users via email when target values are met.
  • User-Friendly Dashboard: View portfolio performance, target values, and notifications.

Technology Stack

Mutual Fund Tracker is built using the following technologies:

  • Frontend: HTML, CSS, AngularJS
  • Backend: [NodeJS,Express,MongoDB]
  • External APIs: [https://api.mfapi.in/mf]
